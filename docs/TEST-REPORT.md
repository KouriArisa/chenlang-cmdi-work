## 说明

在当前 Node 25 环境下，`node:test` 直接承载 HTTP 集成测试会触发运行器内部断言，因此项目采用：

- 单元测试：`node:test`
- 集成测试：显式脚本执行

这样可以保持真实接口验证，同时避免运行器本身的环境缺陷影响结果。
